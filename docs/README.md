## Documentation Conventions

### Code Blocks

Shell commands:
```bash
command --flag argument
```

Configuration files:
```env
KEY=value
```

TypeScript code:
```typescript
function example(): void {
  // Implementation
}
```

### Placeholders

- `your_api_key_here` - Replace with your actual API key
- `yourusername` - Replace with your GitHub username
- `PROJ` - Replace with your Jira project key

### Paths

- Linux/macOS paths: `/path/to/file`
- Windows paths: `C:\path\to\file`
- Relative paths: `./relative/path`

### Command Prompts

- `$` - Regular user command
- `#` - Root/administrator command
- No prompt - Output or file content
